#  ReconX - Advanced Web Recon Automation Toolkit



ReconX is a powerful and automated web reconnaissance toolkit designed for cybersecurity enthusiasts, researchers, and ethical hackers. It streamlines the process of information gathering and attack surface mapping by combining multiple recon techniques into a single workflow.

---

## 🚀 Features

🔍 **Tech Stack Detection**
Identify technologies used by target websites (CMS, frameworks, servers, etc.)

🌐 **Subdomain Enumeration**
Discover hidden and public subdomains to expand attack surface

🟢 **Alive Host Detection**
Filter and detect live hosts for efficient targeting

📂 **Directory & Endpoint Discovery**
Bruteforce and identify hidden directories and endpoints

📜 **JavaScript Analysis**
Extract sensitive endpoints, API keys, and hidden logic from JS files

🕰️ **Wayback + Historical URLs**
Fetch archived URLs to uncover old and forgotten endpoints

🧪 **Parameter Mining**
Discover hidden parameters that may lead to vulnerabilities

📊 **Automated Report Generation**
Generate structured reports for findings and analysis

---

## ⚙️ Workflow

ReconX follows a structured pipeline:

1. Target Input
2. Subdomain Discovery
3. Live Host Filtering
4. Endpoint & Directory Discovery
5. JS + Historical Data Analysis
6. Parameter Extraction
7. Report Generation
